# 🌦️ Atmos

### Open Source Weather Intelligence Platform

Beautiful, fast, privacy-first weather platform for desktop and web.

[Live Demo](#) • [Documentation](#) • [Contributing](CONTRIBUTING.md) • [Report Bug](../../issues)

---

## ✨ Features

- 🌍 Global weather forecasts
- 🛰️ Interactive weather maps
- 🌧️ Real-time precipitation radar
- 🌡️ Hourly & 15-day forecasts
- 🌬️ Air Quality Index (AQI)
- ☀️ UV Index
- 💨 Wind & Pressure
- 🌙 Sunrise, Sunset & Moon Phase
- ⚠️ Severe weather alerts
- 📊 Beautiful analytics dashboards
- 📍 Multiple saved locations
- 🔎 Powerful location search
- 📈 Historical weather data
- 📰 Weather news & climate insights
- 🌎 Climate statistics
- ⚡ Lightning fast
- 🔒 Privacy first
- ❤️ Open Source
